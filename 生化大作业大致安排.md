## 格式要求

### 反应部分（最后尽量整合为一个sheet，命名为"reaction"，以pathway进行区分的唯一标签）
- Pathway
	- glycolysis, gluconeogenesis, etc.
	- 每一个反应的pathway标签是唯一的
		- 对于多个pathway中含有相同的反应（包括反应场所、酶、反应物、生成物，即为同一个反应），麻烦各位再复制一份，标记为另一个pathway
- Description
	- 一段用于描述该反应的文字
	- 如果该反应有什么特殊之处，在此标出
		- 如在mitochondrial membrane上进行，某反应物需要从哪里转运到哪里等
- Multi Step
	- 1表示多步反应，0表示单步反应
- Main Reactant
	- 主要反应物，尽量只写一个（反应链路内的），若有不得不出现多个的情况可以单独提出来商量一下
- Other Reactant
	- 已知的尽量写全
	- 统一格式：角标以正文的形式呈现
		- NAD+
		- H2O
		- CO2
		- ...
- Main Product
	- 可以写多个
- Other Product
	- 同上
- Enzyme
	- 如果有缩写（讲义上明确提出）尽量写缩写
	- 有就写，没有就写0
- Coenzyme
	- 有就写，没有就写0
- Reversibility
	- 1代表可逆，0代表不可逆
- Thermodynamics Effect
	- 1代表吸能（ΔG>0），0代表放能（<0）\*
		- 与reversibility尽量保持一致
- Activator
	- 只写那种简单的regulation，比如说生成物、反应物简单地对其regulate的类型
		- 如ATP抑制的反应，不涉及酶形态变化，可以直接写ATP在inhibitor里
- Inhibitor
	- 同activator
- Species
	- vertebrates
	- plants\*
	- bacteria
	- 统一用复数形式
- Place in Cell
	- cytosol
	- mitochondria
- Special Regulation
	- 如果有单独讲过的regulation机理，请在这一栏单独标注出来，并将regulation对应的图片保存下来
	- allosteric regulation
	- feedback regulation
	- accumulative regulation
	- ...
#### 格式说明
- 对于反应物、生成物（含other），格式统一为
	- **"\name1":"\number1 in reaction";"\name2":"\number2 in reaction"**
	- 如
		- pyruvate:1
		- ADP:1;Pi:1
	- 对于没有提及化学计量数的反应，冒号后用0表示
- 所有物质统一用**全小写**，缩写、特定反应物（名称中的大写含有特殊含义）除外
	- 氨基酸写全称，用全小写：alanine, glutamine, etc.
	- 除外的物质举例：ATP, Pi, CO2, PRPP, PFK-1, dTTP, **acetyl-CoA**
- 所有的没有提及就写**0**或**un**，若使用了其他格式请单独在群里说明
### 其他sheet
#### "special abbreviation"
- abbreviation
	- 写前面reaction中提到的缩写，如PRPP
- full name
	- 写全称，全小写
#### "regulation"（注意只写酶有两种状态的类型）
- Pathway
	- 作为标签
- Enzyme
	- 写酶的缩写，如ATCase
- Active State
	- 如：
		- R state
		- α form (phosphorylated)
- Inactive State
	- 同上
- Activator
	- 命名方式与reaction中的相同
	- 能直接让酶转化为active state的物质
	- 直接抑制酶转化为inactive state的物质
- Inhibitor
	- 同上
- P.S.
	- 附加标注，注明不方便通过activator或inhibitor形式呈现的内容，如
		- ATP is able to prevent the changes induced by CTP
			- 此处ATP并非作为直接的regulator，不方便直接列出，故在此列单独标注
##### 注意
- 尽管有这个单独的sheet，仍然要在reaction中special regulation中标记出来，并且要保存好图片
- 没有的仍然写0处理
- 物质大小写格式同reaction

## 可视化目标
1. 能够显示糖类、脂质、氨基酸、核糖四类物质间的所有代谢路径
	1. 按照反应物->酶->生成物的形式给出，用不同颜色进行区分
	2. 在代谢途径中的反应物使用实线连接，不在代谢途径中的其他反应物使用曲线连接与区分
	3. 能够选中不同的反应途径高亮显示
2. 能够在可视化物种关系的同时，绘制所有可以绘制物的化学结构式
3. 点击反应节点时，能够显示反应节点的详情信息（显示反应物或者酶的详情信息）
4. 能够给出反应物的定量计算关系
